# MOD EKINERJA KOTAWARINGIN BARAT #
-----------------------------------
UPDATE : 1 FEBRUARI 2025

Cara Instalasi:
1. Centang Developer Mode di Browser
2. Centang Allow User Scripts 
   contoh untuk point 1 dan 2
   https://raw.githubusercontent.com/hendrik-svm/mod-ekinerja/refs/heads/main/Developer%20Mode%20dan%20Allow%20User%20Scriptt.png
3. Install Tampermonkey di browser
4. Copy semua isi script yang ada di dalam file script, kemudian masukkan kodenya ke dalam extentions atau add on Tampermonkey yang ada di browser
