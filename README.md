## Instalasi

### 1. Aktifkan Developer Mode dan Allow User Scripts

Sebelum menginstal MOD EKINERJA, aktifkan terlebih dahulu:

- **Developer Mode** pada browser
- **Allow User Scripts**

Panduan pengaturan dapat dilihat pada gambar berikut:

https://raw.githubusercontent.com/hendrik-svm/mod-ekinerja/refs/heads/main/Developer%20Mode%20dan%20Allow%20User%20Scriptt.png

### 2. Instal Tampermonkey

Unduh dan pasang ekstensi Tampermonkey pada browser yang digunakan.

### 3. Pasang Script MOD EKINERJA

1. Buka Tampermonkey.
2. Buat script baru.
3. Salin seluruh isi script dari tautan berikut:

https://raw.githubusercontent.com/hendrik-svm/mod-ekinerja/main/script

4. Tempelkan (Paste) script ke editor Tampermonkey.
5. Simpan script.

## Catatan Penting

> Apabila suatu saat MOD EKINERJA tidak dapat digunakan karena adanya perubahan pada aplikasi e-Kinerja, silakan periksa repositori GitHub ini secara berkala.

Script akan diperbarui apabila ditemukan perubahan sistem atau bug yang menyebabkan fitur tidak berfungsi dengan baik.

Pastikan untuk selalu menggunakan versi script terbaru dengan melihat **tanggal pembaruan (Update)** yang tercantum pada README atau file script.
